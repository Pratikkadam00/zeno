Renders a Lucide icon by kebab-case name. Lucide is Zeno's icon set — 2px stroke, round caps.

```jsx
<Icon name="wallet" size={20} />
<Icon name="chevron-right" color="var(--text-tertiary)" />
```

- Requires `<script src="https://unpkg.com/lucide@latest"></script>` on the page.
- Keep size at 20 for inline UI, 24 for nav/tab bar, 16 for dense rows.
