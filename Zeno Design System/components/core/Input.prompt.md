Single-line text field. Focus ring is the Zeno green at 35% in a 4px halo.

```jsx
<Input label="Subscription name" placeholder="e.g. Netflix" />
<Input label="Monthly cost" prefix="$" mono placeholder="0.00" />
<Input label="Email" error="That doesn't look right" />
```

- Use `prefix="$"` + `mono` for money. 48px tall to stay tappable.
